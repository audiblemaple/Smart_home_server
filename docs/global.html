(async) handleControlCommand(req, res)

This function reads an 'action' and 'node' from the request body. Based on the action ('open', 'close', 'stop'), it performs the corresponding operation. If the action is invalid, it sends a 400 response with an 'Invalid action' message. In case of successful execution of the action, it sends a confirmation message. Errors are caught and logged, and an 'Internal Server Error' message is sent.
Parameters:
Name Type Description
req Object The request object, containing the action and slot in its body.
